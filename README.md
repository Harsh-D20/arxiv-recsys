# Setup

1. Create a venv with the command:
   ```bash
   python -m venv venv
   ```

2. Activate the venv:
   ```bash
   source venv/bin/activate
   ```

3. Install the requirements from requirements.txt:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the project:
   ```bash
   python main.py
   ```
